O SI foi implementado como um Aplicativo para plataformas iOS, no momento de entrega do trabalho 
o aplicativo está passando por revisões na app store e, se tudo ocorrer como planejado, no momento da 
correção ele ja estará disponível para download como HazeTasks. Essa seria a maneira mais adequada de testar.

Para execução do aplicativo sem que necessite baixa-lo da App Store é necessário de um computador que esteja rodando
a versão mais recente do macOS e possua baixado o XCode. Para abrir o projeto no XCode basta acessar o diretório /RenatoNobre_150146698/prototipo/Arquivo Completo XCode/HazeTasks e executar o arquivo HazeTasks.xcodeproj, dentro da plataforma
de desenvolvimento, escolha algum dispositivo para teste, (recomendado iPhone SE) e execute o código.

Caso não seja possível o acesso a plataforma XCode os códigos desenvolvidos estão contidos na pasta prototipo/Códigos 

A linguagem de programação utilizada foi a Swift 3.0, a linguagem POO mais nova de desenvolvimento para Apple.
https://swift.org/blog/
